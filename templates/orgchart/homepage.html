EmployeeWare: Open Source HR Software

Use the software

  • Browse List of Employees
    • There you can click on any employee to start browsing the org chart.
  • Administration Interface
    • This takes you to the tools to manage the org chart, i.e. add/change employees and reporting relationships.

About this software

  • Test drive this software
    • To play with, practice and learn how to create and edit an org chart using this tool, go to the demonstration site at demo.employeeware.org/admin and login in using the username 'demo' and password 'demo' (without the quotes).
  • EmployeeWare Project: Get this Software
    • There you can download this software to run a copy on your own servers.
    • You can also submit suggestions, feature requests and issues.
    • If you are a software engineer, designer or product developer, you are also welcome to get involved in the project.
      • This software was built using the Python Programming Language, the Django Web Application Framework and the Google Chart Tools APIs in JavaScript & HTML5.
      • The demo application and project are hosted at the Atlassian bitbucket and the Amazon Web Services (AWS).
